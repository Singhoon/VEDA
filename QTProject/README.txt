# 심화 실습 평가1(Qt) 최종 제출
### 프로그램명 : VEDA 게임 천국
#### 제작자 : 김태훈, 조수연
## 프로그램 실행 방법
* 개발 환경은 Qt creater를 사용하였고, creater를 이용해 실행하면 됩니다.
* resource 파일들과 코드들을 같은 폴더에 위치하게 해야합니다. 해당 파일은 같이 압축되어 있으므로 CMakeLists.txt를 통해 그대로 빌드하면 됩니다.
## 주의사항
* resource 파일이 코드가 있는 곳에 위치하지 않으면 이미지가 제대로 표시되지 않을 수 있습니다.
## 메인 화면
1. 게임 선택, 프로필, 종료 메뉴가 있습니다.
2. 게임을 선택하기 위해서는 프로필을 선택해야 합니다. 프로필은 버튼을 눌러 직접 들어가거나, 프로필이 없을 때 게임 선택 버튼을 누르면 다이얼로그를 통해 이동할 수 있습니다.
3. 프로필을 만든 후 게임 선택을 누르면 현재 2개의 게임을 선택하거나, 다시 메인화면으로 돌아가는 버튼이 있습니다.
4. 처음 프로그램이 실행될 시, 만약 저장된 프로필이 있다면 자동으로 불러와집니다.
## 프로필
1. 프로필 생성 : 프로필 생성 버튼을 통해 프로필을 작성할 수 있습니다. <br>
프로필을 생성하면 기본적으로 두 게임의 최고 점수를 기록하는 프로필이 생성됩니다. <br>
프로필이 생성될 때 해당 프로필이 자동으로 선택됩니다. 클릭을 통해 프로필을 변경할 수 있습니다.
2. 프로필 삭제 : 프로필 삭제 버튼을 통해 프로필을 삭제할 수 있습니다. <br>
아무 프로필이 없다면 작동하지 않고, 현재 선택된 프로필이 있다면 다이얼로그를 통해 삭제 및 취소가 가능합니다.
3. 프로필 검색 : 프로필 검색 버튼을 통해 프로필을 검색할 수 있습니다. <br>
다이얼로그를 통해 이름으로 검색할 수 있고, 가장 위쪽에 있는 프로필이 선택됩니다. 해당 이름이 없으면 다이얼로그를 통해 없다는 알림을 보냅니다.
4. 프로필 수정 : 프로필 수정 버튼을 통해 프로필을 수정할 수 있습니다.<br>
선택된 프로필의 이름을 변경할 수 있습니다.
5. 프로필 화면에서 메뉴로 돌아가기를 누를 시 프로필은 자동으로 저장됩니다. 또한 게임에서 메인 화면으로 돌아올 때도 자동으로 저장됩니다.
## 2048
* 2048 게임은 4X4 보드에서 같은 숫자가 적힌 두 타일을 합쳐 최종적으로 '2048'이 적힌 타일 하나를 만드는 것입니다.
* 패배 (게임 오버): 보드의 16칸이 모두 타일로 가득 차 더 이상 합칠 수 있는 타일이 없거나, 어떤 방향으로도 타일을 이동시킬 수 없게 되면 게임에서 패배합니다.
* 시작은 2 또는 4 타일 2개를 가지고 시작합니다.
* 방향키를 통해 게임을 진행할 수 있습니다.
* undo 버튼을 통해 타일을 한차례 뒤로 되돌릴 수 있습니다. undo 버튼은 총 2번 사용 가능하며, 한 움직임당 한번 사용 가능합니다 (2회 연속 불가능)
* new 버튼을 통해 게임을 새로 시작할 수 있습니다.
* menu 버튼을 통해 메뉴로 돌아갈 수 있습니다 메뉴로 돌아가면 해당 게임에서 얻은 max 점수를 자동으로 프로필에 등록합니다.
* gameover 시 새로운 게임을 만들거나 메뉴로 돌아갈 수 있고, undo 버튼이 남아있다면 다이얼로그를 통해 한차례 이전으로 되돌릴 수 있습니다. 없다면 메뉴나 새로운 게임을 만들 수 있습니다.
## Yatzee
* Yatzee는 5개의 주사위를 굴려 특정 조합(족보)을 만들고, 총 13라운드 동안 점수판을 채워 가장 높은 점수를 획득하는 전략 게임입니다.
* 각 라운드마다 주사위를 최대 3번까지 굴릴 수 있습니다.
* 주사위를 클릭하여 원하는 주사위를 보관(Keep)하고 나머지 주사위만 다시 굴릴 수 있습니다. 보관된 주사위는 빨간색 테두리로 시각적으로 구분됩니다.
* 주사위를 굴린 후, 점수판의 'Potential' (예상 점수) 열을 클릭하여 점수를 확정합니다. 점수 기록이 가능한 칸은 텍스트가 붉은색으로 강조됩니다.
*점수가 0점인 항목을 선택할 경우, 실수로 기록하는 것을 막기 위해 **재확인 경고창(Warning Dialog)**을 띄웁니다.
* 롤 횟수가 남아있을 때 모든 주사위를 Keep 상태로 두면, 불필요한 턴 소모를 막기 위해 'Roll Dice' 버튼이 비활성화됩니다.
* 라운드 시작 시 'Roll Dice' 버튼이 깜빡이며 사용자의 행동을 유도합니다.
* 13라운드가 종료되면 게임 결과창이 나타납니다.
1. Retry: 게임을 즉시 초기화하고 다시 시작합니다.
2. Quit Game: 메인 메뉴로 돌아가며, 현재 점수가 프로필의 최고 점수보다 높을 경우 자동으로 갱신됩니다.
*'Rules' 버튼을 통해 언제든지 게임의 규칙(상단/하단 섹션 점수 계산법)을 확인할 수 있습니다.
